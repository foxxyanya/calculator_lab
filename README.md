# calculator_lab

Python version - 3.8

### Installation guide:

1. Create and activate python env:

   ```
   python3.8 -v venv env
   source env/bin/activate
   ```
2. Install requirements:

   ```
   pip install --upgrade pip
   pip install -r requirements.txt
   ```
3. Run app:

   ```
   python main.py

   ```
